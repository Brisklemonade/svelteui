## Usage

<p>
	The <strong>{meta.title}</strong> action calls given function when the current tab is switched
</p>

```tsx
<script>
    import { tableave } from '@svelteuidev/actions';

    $: count = 0;
</script>

<div use:tableave={() => count++}>{count}</div>
```

## Params

| Param    | Description              |
| -------- | ------------------------ |
| callback | The callback to be fired |

## Events

<p>
	The <strong>{meta.title}</strong> action does not dispatch any custom events`
</p>

```ts
// No events to display
```

## Definition

```ts
export function tableave(node: HTMLElement, callback: Fn<void>): ReturnType<Action>;
```

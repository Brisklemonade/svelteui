## Usage

<p>
	The <strong>{meta.title}</strong> action calls given function when mouse leaves the page
</p>

```tsx
<script>
    import { pageleave } from '@svelteuidev/actions';

    $: count = 0;
</script>

<div use:pageleave={() => count++}>{count}</div>
```

## Params

| Param    | Description              |
| -------- | ------------------------ |
| callback | The callback to be fired |

## Events

<p>
	The <strong>{meta.title}</strong> action does not dispatch any custom events`
</p>

```ts
// No events to display
```

## Definition

```ts
export function pageleave(node: HTMLElement, callback: Fn<void>): ReturnType<Action>;
```

## Usage

`lockscroll` locks scroll at current position by setting document.body overflow to hidden

<Demo demo={ComposableDemos.useLockScrollDemo.usage} />

## Params

| Param | Description                                            |
| ----- | ------------------------------------------------------ |
| lock  | A boolean to specify whether or not to lock the screen |
